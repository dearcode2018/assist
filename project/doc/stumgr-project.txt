【项目信息】
项目名称 : stumgr
中文名称 : 学生管理系统
项目版本 : version 1.0

主要技术 : 
前端部分 : html5、css 3.x、javascript、jquery 2.x、jquery easy ui、、、、
服务端 : Servlet 3.x、 myBatis 3.x、Spring 3.x、Jdbc 3.x、log4j、poi、mail、quartz、jackson
、memcached、hibernate 4.x、Spring mvc、、、、
部署部分 : linux OS、tomcat 7.x、集群/负载均衡、jboss、、

项目描述 : 学生管理系统，包含学生信息、课程管理、选课管理、邮件验证和通知、
后台管理、批量导入、批量导出、任务调度、用户登录、用户注册、修改密码、用户管理、用户过期处理、
文件上传(图片/文档...)、开关功能(相关的功能尽量采用开关机制)、

项目备注 : 
资源目录规划(公共文件区、用户文件区-每个用户对应一个文件夹)、数据表设计与规划、数据交互(json格式)
b/s架构，目前规划只开发web端，其他端暂不规划.

【项目规划】
1) 用户管理
用户注册、用户登录、修改密码、用户有效/无效/禁用、用户信息修改、头像上传、
功能点 : 注册、登录、注销、修改密码、验证码、邮箱验证和激活、图片上传、
用户类型 : 系统管理员、普通管理员、学生、教师、教导员、

2) 后台管理
批量导入、导出，任务调度、定时扫描、群发邮件、单发邮件、

3) 课程管理
课程增删改查、课程数据表设计、课程划分、课程列表和详情、

4) 选课管理
课程列表查询、学生选课、选课结果、多用户选择、表锁定、

5) 优化1、2期的开发

6) 优化3、4期的开发

7) 修改bug
